# ppf-calculator
This PPF Calculator is created using HTML5/CSS3 and Javascript that calculates the maturity amount and interest earned on the PPF account for 15 years of investment based on user input of yearly investment. Checked validation on the input of yearly investment field (like the min. and max. investment amount and empty input). Showed results in the table containing columns like Year of Investment, Opening Balance, Amount Deposited, Interest Earned, Closing Balance, Total Amount Deposited and Maturity Amount after 15 years.
